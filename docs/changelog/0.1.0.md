# Changelog - Version 0.1.0

**Release Date**: 2024-XX-XX
**Status**: Alpha

## Added

### Project Management

-   `fastappkit core new` - Create new FastAPI projects
-   `fastappkit core dev` - Run development server

### App Management

-   `fastappkit app new` - Create internal apps
-   `fastappkit app new --as-package` - Create external apps
-   `fastappkit app list` - List all apps in project
-   `fastappkit app validate` - Validate app structure and manifest

### Migration Management

-   `fastappkit migrate core` - Create core migrations
-   `fastappkit migrate app <name> makemigrations` - Create app migrations
-   `fastappkit migrate app <name> upgrade` - Upgrade app migrations
-   `fastappkit migrate app <name> downgrade` - Downgrade app migrations
-   `fastappkit migrate app <name> preview` - Preview migration SQL
-   `fastappkit migrate all` - Run all migrations
-   `fastappkit migrate preview` - Preview all migrations
-   `fastappkit migrate upgrade` - Upgrade all migrations
-   `fastappkit migrate downgrade` - Downgrade all migrations

### Core Features

-   App discovery and loading (internal and external)
-   Automatic router mounting with configurable prefixes
-   App manifest system
-   App validation (isolation, manifest, migrations)
-   Migration ordering (core → internal → external)
-   Template rendering system (Jinja2)

### Configuration

-   `fastappkit.toml` configuration file
-   Environment variable support via `.env`
-   Settings protocol for type-safe configuration
