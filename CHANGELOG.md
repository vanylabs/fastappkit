# Changelog

All notable changes to fastappkit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## Version History

-   **[0.2.1](docs/changelog/0.2.1.md)** - Fixed migration system to prevent dropping external app tables
-   **[0.2.0](docs/changelog/0.2.0.md)** - Enhanced external app support and configuration improvements
-   **[0.1.9](docs/changelog/0.1.9.md)** - Documentation improvements
-   **[0.1.0](docs/changelog/0.1.0.md)** - Initial alpha release

## Release Types

-   **MAJOR** (X.0.0) - Breaking changes

    -   App manifest format changes
    -   Migration system changes
    -   CLI command structure changes
    -   Template structure changes

-   **MINOR** (0.X.0) - New features

    -   New CLI commands
    -   New configuration options
    -   New app features
    -   Backward-compatible changes

-   **PATCH** (0.0.X) - Bug fixes and security patches
    -   Bug fixes
    -   Security patches
    -   Documentation updates
    -   Performance improvements

## Migration Guides

When breaking changes are introduced, migration guides will be provided in the release notes and documentation.

For detailed changelog entries, see version-specific files in [docs/changelog/](docs/changelog/).

---

**Note**: Detailed changelog entries for each version are maintained in separate files in `docs/changelog/` for better organization and maintainability.
